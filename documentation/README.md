# documentation Folder
This folder contains hand-maintained project notes and ontology design documents. Unlike `docs/`, which contains generated website output, the files here are working documentation used during ontology development.

### Important Contents
- `alignment-module.md` - Notes about ontology alignment work for modules.
- `competency-questions.md` - Competency questions used to validate the ontology.
- `currKG-axioms.owl` - OWL axioms file stored alongside the documentation set.
- `key-notions.md` - Core concepts and terminology for CurriculumKG.
